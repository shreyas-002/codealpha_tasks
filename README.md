### Project 2: Cryptocurrency Wallet Interface

A frontend cryptocurrency wallet interface designed to simulate wallet functionality using client-side state.

**Key highlights:**
- Wallet dashboard with supported cryptocurrencies
- Balance tracking using mock data
- Transaction history view
- Send and receive UI (simulation only)
- No real blockchain or backend integration

## Running Projects Locally

Each project contains its own README file with detailed instructions on how to run it locally.

General steps:
1. Navigate to the specific project folder
2. Install dependencies using `npm install`
3. Start the development server using `npm run dev`
